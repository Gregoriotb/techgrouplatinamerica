# SC-DESIGN-TGLA | UI/UX Design System

## Layout Landing Page (International Brochure)
1. **Hero Global:** Mapa estilizado América (USA + LATAM) con líneas de conexión animadas. H1: "Su puente tecnológico entre USA y Latinoamérica"
2. **Route Visualization:** Gráfico interactivo simple: USA Flag → [Avión/Barco] → LATAM Flags (México, Colombia, Chile, Perú, etc.)
3. **Servicios Grid:** 
   - Importación Tech
   - Exportación Alimentos  
   - Logística Integral
   - Consultoría Aduanera
4. **Cobertura:** Mapa con pins en ciudades clave + lista de países donde operan
5. **Ventajas Competitivas:** Cards con iconos: "Tiempo de tránsito", "Costos competitivos", "Cumplimiento aduanero", "Soporte bilingüe"
6. **Contacto Internacional:** Selector de país + formulario

## Componentes Específicos
- Nav: Logo + "Tech Group" / "Latin America" en dos pesos tipográficos, selector de idioma (ES/EN)
- Hero: Canvas o SVG con líneas animadas conectando puntos
- Cards servicio: Border-left colorido según tipo (Azul=Tech, Verde=Food, Naranja=Logística)
- Tipografía: Poppins (moderna, geométrica) para todo
